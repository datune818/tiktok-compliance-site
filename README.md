# TikTok Compliance Placeholder Site
This static website exists only to provide Privacy Policy and Terms of Service URLs required by TikTok Developer Portal.

To deploy on GitHub Pages:
1. Commit these files to a public GitHub repository (e.g., tiktok-compliance-site).
2. Go to repository Settings → Pages → set Source = “main” branch, / (root).
3. Wait 1–2 minutes; the site will be available at:
   https://<your-username>.github.io/tiktok-compliance-site/
4. Copy:
   - https://<your-username>.github.io/tiktok-compliance-site/privacy.html  → Privacy Policy URL
   - https://<your-username>.github.io/tiktok-compliance-site/terms.html    → Terms of Service URL
   and paste them into the TikTok Developer Portal.
